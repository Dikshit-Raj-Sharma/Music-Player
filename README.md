# Web-Based Music Player

This project is a web-based audio player application built using HTML, CSS, and JavaScript. It provides a clean and modern user interface for playing music tracks directly in the browser.

---

## Features

* **Media Controls:** Standard controls including play, pause, next track, and previous track.
* **Audio Playback:** Displays song details like cover art, title, and artist.
* **Progress Bar:** A visual progress bar that shows the current playback time and allows seeking.
* **Volume Control:** A slider to adjust the audio volume.
* **Playback Options:** Additional controls such as shuffle and repeat for the playlist.
* **Track List:** A view to see the list of available songs.

---

## Technologies Used

* **HTML5**: For the core structure and markup of the application.
* **CSS3**: For styling the user interface, layout, and responsiveness.
* **JavaScript (ES6+)**: For handling all application logic, audio playback, and user interactions.

---

## How to Use

1.  Ensure you have all the project files (HTML, CSS, JavaScript, and audio/image assets).
2.  Open the `index.html` file in any modern web browser.
3.  Use the on-screen controls to play music.
